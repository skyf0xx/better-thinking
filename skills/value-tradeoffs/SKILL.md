# Value Tradeoffs

Make competing values explicit and price the actual exchange rate between them, rather than letting a decision silently smuggle in one value's priority over another.

## Why

Real decisions routinely trade one genuine value against another — speed versus safety, autonomy versus equity, privacy versus transparency — but this trade is often made implicitly, without anyone deciding on purpose what the exchange rate should be. Making it explicit turns a hidden default into a deliberate, defensible choice.

## Use when / Don't use when

- **Use when:** a decision keeps stalling because "it depends what you prioritize"; two people disagree despite agreeing on the facts; a decision has an uncomfortable tradeoff being avoided rather than named.
- **Don't use when:** the values genuinely don't conflict in this case — check first before assuming a tradeoff exists.

## Inputs → Outputs

- **Inputs:** a decision where two or more legitimate values conflict.
- **Outputs:** the competing values named explicitly, the actual exchange rate the decision implies, and a deliberate judgment on whether that rate is the right one.

## Principles

- Name both values precisely — vague value labels like "fairness" or "innovation" hide the actual conflict.
- State the exchange rate the decision implies as concretely as possible — pricing it, even roughly, is what makes the tradeoff visible instead of assumed.
- A values disagreement can't be resolved by more evidence — recognizing that early prevents an unproductive argument that pretends to be about facts.

## Procedure

1. Identify the two or more values genuinely in tension for this specific decision.
2. State each precisely — not as a slogan, but as what it actually requires in this context.
3. Estimate the exchange rate the leading option implies: how much of value A is given up for how much of value B?
4. Ask whether that rate is the one you'd endorse on reflection, or whether it's arrived at by default rather than judgment.
5. Where people disagree, check whether the disagreement is about facts or about the exchange rate itself.
6. Report the tradeoff and the judgment made, so it's visible and revisitable rather than buried.

## Common mistakes

- Treating a values disagreement as if more evidence would resolve it.
- Leaving the exchange rate implicit, so the decision silently favors whichever value the loudest voice in the room happens to prioritize.
- False-precision pricing a genuinely incommensurable tradeoff.

## Examples

- A product's speed-to-market versus quality tradeoff, priced explicitly rather than assumed.
- A policy balancing individual privacy against collective safety.
- A personal choice between career advancement and family time.
